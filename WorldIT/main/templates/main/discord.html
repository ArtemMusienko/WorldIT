Discord
Discord — кроссплатформенная проприетарная система мгновенного обмена сообщениями с поддержкой VoIP и видеоконференций, предназначенная для использования различными сообществами по интересам. Наиболее популярна у геймеров и учащихся. Разработчиком является компания Discord Inc. из Сан-Франциско.

Discord чат-боты

Сейчас они становятся все более и более актуальными. Почти каждый Discord сервер оснащен ботами.

Почему именно Discord чат-боты?

  • Востребованность и нарастающая популярность - одно из главных качеств платформы Discord
  • Многообразие категорий у чат-ботов не обходят Discord стороной, развивая его быстрыми темпами
  • Возможность быстро развивать свой проект в Discord с помощью чат-ботов
  • С помощью них можно организовать лучшие условия для игры и общения

Как использовать?

Нажмите на кнопку ниже и прочтите подробную инструкцию в использовании Discord чат-бота

Инструкция для Discord ботов

  • 1. После успешной покупки Вашего Discord бота переходим к содержимому покупки.
  • 2. После создания Вашего сервера на Discord перейдите к загрузке бота.
  • 3. На вкладке Applications выберите New Application.
  • 4. Введите название будущего приложения и нажмите Create.
  • 5. Приложение создано. Перейдите не вкладку Bot и нажмите Add Bot, чтобы добавить нового бота.
  • 6. Бот создан. На вкладке Bot отобразится вся информация о нем. Тут можно изменить его имя, добавить изображение и скопировать токен бота. Этот токен понадобится вам для настройки модуля Discord на платформе Appmaster.i. Каждый раз при внесении изменений портал разработчика будет предлагать вам сохранить их. Сохраните токен бота, чтобы при последующей настройке не возвращаться к этому шагу.
  • 7. Теперь перейдите на вкладку OAuth2 — тут можно настроить разрешения и получить ссылку на вашего бота. В разделе SCOPES выберите bot, в BOT PERMISSIONS отметьте разрешения, которые хотите ему предоставить, в нашем случае — только отправка сообщений, поэтому выберите Send Message. После скопируйте автоматически сгенерированную Discord ссылку.
  • 8. Вставьте скопированную ссылку в адресную строку браузера и перейдите по ней — откроется окошко вашего приложения. Выберите ваш сервер в раскрывающемся списке и нажмите Continue.
  • 9. Убедитесь, что приложению (а значит, и созданному в нем боту) даны только необходимые разрешения, после чего добавьте его на ваш сервер.
